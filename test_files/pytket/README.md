# Example pytket circuits

This directory contains pytket circuits used for testing and benchmarking.

### Circuit descriptions

| Name                     | Description
| ------------------------ | -------------------------------------------------
| barenco_tof_5            | Barenco decomposition of a 5-qubit Toffoli into Clifford+T
| barenco_tof_10           | Barenco decomposition of a 10-qubit Toffoli into Clifford+T
